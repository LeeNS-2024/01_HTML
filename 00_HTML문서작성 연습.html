목록 관련 태그

ul 태그 : 순서가 없는 목록 (Unordered List)

li 태그 : 목록의 한 줄 (List Item)

ol 태그 : 순서가 있는 목록 (Ordered List) 

    - type 속성 : 표시되는 숫자 형식 지정

    type="a" : 영어 소문자(a,b,c,d,...)
    type="A" : 영어 대문자(A,B,C,D,...)
    type="i" : 로마 소문자(i,ii,...,v,...,x)
    type="I" : 로마 대문자(I,II,...,V,...,X)
    
    start 속성 : 목록 시작 번호

    reversed 속성: 순서 뒤집기(역순, 시작부터 -1씩)

Emmet : HTML, CSS 빠른 자동완성 지원 플러그인

    태그명 : 태그 자동 완성
    >  : 태그 내부에 작성 (하위)
    +  : 태그 옆에 작성 (형제)
    *@ : @만큼 생성 (곱하기)
    {} : 태그 내부에 작성될 내용
    $  : 1부터 1씩 증가하는 숫자

표 관련 태그


  1. table 태그
    - 웹 문서에서 자료(데이터)를 정리하기 위해 사용하는 태그
    - table 태그는 행, 열을 작성하기 위한 판을 지칭하는 태그
  2. tr 태그 (Table Row)
    - 테이블의 한 행을 나타내는 태그
  3. td 태그 (Table Date)
    - 테이블 데이터를 작성할 수 있는 태그
    - 각 행(tr)의 열(column) 역할을 함 == 엑셀의 한 셀(한 칸)
  4. th태그 (Table Header)
    - 테이블의 제목 또는 내용을 강조할 때 td대신 사용하는 태그
    - 굵은 글씨 + 가운데 정렬
  5. caption 태그
    - 테이블에 대한 설명을 작성하는 태그
  6. thead 태그
    - 테이블 상단 영역(컬럼 제목 영역)
  7. tbody 태그
    - 테이블 중단 영역(값 작성 영역)
  8. tfoot 태그
    - 테이블 하단 영역(합계, 총합 영역)
